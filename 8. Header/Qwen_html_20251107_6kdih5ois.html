Responsive Sticky Header

Features:

  • Sticky header with smooth shadow effect
  • Dark/light mode toggle
  • Responsive design with mobile hamburger menu
  • Multi-level dropdown navigation
  • Off-canvas mobile menu with nested items
  • Custom CSS variables for easy theming

Desktop View

The header displays the logo, navigation links, and CTA button in a single row. Hover over "Products" to see the dropdown menu.

Mobile View

On smaller screens, the navigation collapses into a hamburger menu. Click the menu icon to open the off-canvas navigation.

Scroll Down

The header will stick to the top when you scroll
